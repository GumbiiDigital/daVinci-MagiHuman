20 CTAs organized by platform and urgency level. Copy-paste ready.

Replace [PRODUCT], [PRICE], and [LINK LOCATION] as needed.

## By Platform

### TikTok CTAs

TikTok rewards fast, casual, low-friction language. No polished sales energy. Talk like you're texting a friend.

| Urgency | CTA |
|---------|-----|
| Soft | "Anyway, it's like [PRICE] bucks. Do with that what you will." |
| Soft | "Just thought you should know this exists." |
| Medium | "It's [PRICE] dollars. Just try it." |
| Medium | "Get one. Trust me on this." |
| Hard | "Stop scrolling and click the link. You'll thank me later." |
| Hard | "Don't think about it. [PRICE] dollars. Link. Go." |

**TikTok-specific notes:**
- "Link in bio" works for organic. For TikTok Shop, use "Tap the product link" or "Check the orange basket."
- Keep CTAs under 3 seconds. Fast cut to end screen or text overlay.
- Never say "subscribe" or "follow" in the ad CTA -- that's a separate objective.

### Instagram Reels CTAs

Instagram audiences respond to slightly more polished language but still casual. "Link in bio" is the native CTA.

| Urgency | CTA |
|---------|-----|
| Soft | "I'll leave the link in my bio. No pressure." |
| Soft | "Save this for later if you need it." |
| Medium | "Link in bio. You won't regret it." |
| Medium | "I'm telling everyone about this. [PRICE] bucks. Link in bio." |
| Hard | "Link in bio. Go. Right now. Before it sells out again." |
| Hard | "Every day without [PRODUCT] is a day you're dealing with [PROBLEM] for no reason. Link in bio." |

**Instagram-specific notes:**
- Always say "link in bio" not "link below" (IG doesn't have clickable caption links for most accounts).
- "Save this" is a strong CTA because saves boost algorithm ranking.
- Stories swipe-up / link sticker is the highest-converting Instagram CTA format -- repurpose Reels into Stories with link sticker.

### YouTube Shorts CTAs

YouTube audiences expect slightly more information. They tolerate a 5-second CTA better than TikTok viewers.

| Urgency | CTA |
|---------|-----|
| Soft | "If you're curious, I got mine from the link in the description." |
| Soft | "I'll put the link below if you want to check it out." |
| Medium | "I wish someone told me about this sooner. Link in the description." |
| Medium | "Worst case you're out [PRICE] dollars. Best case it changes your [ROUTINE]. Link below." |
| Hard | "Tap the link in the description and grab one while it's still [PRICE] dollars." |
| Hard | "I've bought three of these already. Link in the description, go get yours." |

**YouTube-specific notes:**
- Say "description" not "bio" (YouTube doesn't have bio links in Shorts the same way).
- Pinned comments with links convert well on Shorts -- mention "check my pinned comment" as an alternative CTA.
- YouTube audiences are more research-oriented. "Check the link for the full review" works as a bridge CTA.

## CTA Selection Matrix

Pick urgency based on funnel position:

| Audience Temperature | Urgency Level | When to Use |
|---------------------|---------------|-------------|
| Cold (never seen you) | Soft | First organic post, broad targeting, awareness |
| Warm (engaged before) | Medium | Retargeting, second video in series, comments asked for link |
| Hot (ready to buy) | Hard | Sale/promo, retargeting clickers, cart abandoners |

## Text Overlay CTA Formats

```

```

```
$[PRICE] | [PRODUCT NAME]
```

```
The [PRODUCT] that changed my [ROUTINE]
[PRICE] dollars | link below
```

```

[STAR RATING] | [NUMBER] reviews

```
